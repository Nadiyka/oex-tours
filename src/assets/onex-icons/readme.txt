/**
 * Инструкция по сборке/доработки шрифта иконок "onex-icons":
 */

В этом документе предствлено описание файлов в этой директории.
**-----------------------------------------------------------------**

1. onex-icons.eot - шрифт
2. onex-icons.svg - шрифт
3. onex-icons.ttf - шрифт
4. onex-icons.woff - шрифт
5. selection.json - файл-конфигурация шрифта, который можно загрузить на сервис сборки шрифта и
                    изменить/дополнить/удалить иконки из шрифта.


Как изменить/дополнить или удалить иконки из шрифта:
1. Переходим на сервис https://icomoon.io/;
2. Регистрируемся;
3. Загружаем (импортируем) в сервис файл selection.json, ждем окончания загрузки;
4. Далее добавляем/изменяем или удаляем иконку. (Добавлять иконку(и) необходимо в конец списка иконок).
5. При сохранении шрифта необходимо указать его имя ("onex-icons"). Скорее всего у вас это поле будет заполнено.
6. Затем, после распаковки загруженного архива, извлекаем файлы шрифта и файл-конфигурации (см. перечень файлов) и
   заменяем ими файлы в папке fonts/onex-icons.
7. Далее открываем файл style.css из распакованного архива шрифта и копируем стили иконок и помещаем их в файл css/icons.css.
